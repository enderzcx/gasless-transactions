# ReflectTrade

GitHub User: Uoghluvm
ProjectName: ReflectTrade
Track: LLM 应用赛道
ProjectDescription: ReflectTrade 是一款拦截冲动交易的浏览器插件。
当你在交易下单时，ReflectTrade 会强行介入。基于 SpoonOS 构建的 Graph Agent 会调用 实时链上数据（如资金费率、巨鲸流向），对你的开仓逻辑进行毒舌审计。如果你的理由只是“牛回速归”，AI 会直接拆穿你的幻觉。另外，用户可通过 Kite AI 支付一笔“清醒费”来解锁深度审计报告并完成链上反思存证，公开处刑。
